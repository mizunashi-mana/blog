ハイティング代数の性質

Posted on 日 04 10月 2020 in 数学

古典論理の意味論がブール代数によって議論できるように，直観主義論理の意味論はハイティング代数によって考えることができる．ハイティング代数とは，冪で閉じている有界束であり，ブール代数もハイティング代数になる．

というわけで結構題材になるハイティング代数だが，巷でハイティング代数に関しての資料が少なかったりするので，自分のために諸々備忘録を残しておく．

ハイティング代数の定義

まずは基本的なところから。最初に poset を導入する。

定義. 半順序 (partial order, poset)

集合 AAA とそれ上の二項関係 (⊑)⊆A×A(\sqsubseteq) \subseteq A \times A(⊑)⊆A×A の組 (A,⊑)(A, \sqsubseteq)(A,⊑) で以下を満たすものを、半順序と呼ぶ:

反射律 (reflexivity)
任意の x∈Ax \in Ax∈A について、x⊑xx \sqsubseteq xx⊑x
推移律 (transitivity)
任意の x,y,z∈Ax, y, z \in Ax,y,z∈A について、x⊑yx \sqsubseteq yx⊑y かつ y⊑zy \sqsubseteq zy⊑z ならば、x⊑zx \sqsubseteq zx⊑z
反対称律 (anti-symmetric)
任意の x,y∈Ax, y \in Ax,y∈A について、x⊑yx \sqsubseteq yx⊑y かつ y⊑xy \sqsubseteq xy⊑x ならば、x=yx = yx=y

束の定義方法は色々あるが、今回は poset ベースの束の定義を導入する。

定義. 束 (lattice)

半順序 (A,⊑)(A, \sqsubseteq)(A,⊑) で以下を満たすものを、束と呼ぶ:

  • 任意の x,y∈Ax, y \in Ax,y∈A について、上限 x∨y∈Ax \lor y \in Ax∨y∈A が存在する。この時、x∨yx \lor yx∨y を結び (join) と呼ぶ
  • 任意の x,y∈Ax, y \in Ax,y∈A について、下限 x∧y∈Ax \land y \in Ax∧y∈A が存在する。この時、x∧yx \land yx∧y を交わり (meet) と呼ぶ

束は、幾つかの代数的性質が成り立つ。

定理. 束の性質

束 (L,⊑)(L, \sqsubseteq)(L,⊑) について、以下が成り立つ:

交換律 (commutative)
任意の x,y∈Lx, y \in Lx,y∈L について、x∧y=y∧xx \land y = y \land xx∧y=y∧x、x∨y=y∨xx \lor y = y \lor xx∨y=y∨x
結合律 (associative)
任意の x,y,z∈Lx, y, z \in Lx,y,z∈L について、x∧(y∧z)=(x∧y)∧zx \land (y \land z) = (x \land y) \land zx∧(y∧z)=(x∧y)∧z、x∨(y∨z)=(x∨y)∨zx \lor (y \lor z) = (x \lor y) \lor zx∨(y∨z)=(x∨y)∨z
吸収律 (absorption)
任意の x,y∈Lx, y \in Lx,y∈L について、x∧(x∨y)=x∨(x∧y)=xx \land (x \lor y) = x \lor (x \land y) = xx∧(x∨y)=x∨(x∧y)=x
冪等律 (idempotent)
任意の x∈Lx \in Lx∈L について、x∧x=x∨x=xx \land x = x \lor x = xx∧x=x∨x=x

証明:

交換律，結合律，冪等律はいいと思うので，吸収律だけ示しておく．といっても，こいつもほぼ自明で，x≤x∨yx \leq x \lor yx≤x∨y，x∧y≤xx \land y \leq xx∧y≤x より，下限，上限の定義から x∧(x∨y)=xx \land (x \lor y) = xx∧(x∨y)=x，x∨(x∧y)=xx \lor (x \land y) = xx∨(x∧y)=x．

束の中で、空集合に対する上限、下限が存在するもの、つまり最大元、最小元が存在するものを有界束と呼ぶ。

定義. 有界束 (bounded lattice)

束 (L,⊑)(L, \sqsubseteq)(L,⊑) で以下を満たすものを，有界束と呼ぶ:

  • ある ⊤∈L\top \in L⊤∈L が存在し、任意の x∈Lx \in Lx∈L について x≤⊤x \leq \topx≤⊤ を満たす
  • ある ⊥∈L\bot \in L⊥∈L が存在し、任意の x∈Lx \in Lx∈L について ⊥≤x\bot \leq x⊥≤x を満たす

ハイティング代数は、有界束の中で冪対象を持つものである。

定義. ハイティング代数 (heyting algebra)

有界束 (L,⊑)(L, \sqsubseteq)(L,⊑) で以下を満たすものを、ハイティング代数と呼ぶ:

  • 任意の x,y∈Lx, y \in Lx,y∈L について、{z∈L∣x∧z≤y}\{z \in L \mid x \land z \leq y\}{z∈L∣x∧z≤y} の上限 x→y∈Lx \to y \in Lx→y∈L が存在する。この時、x→yx \to yx→y を冪 (exponential) または相対擬補元 (relative pseudo-complement) と呼ぶ

証明は省略するが，冪の例は x→x=⊤x \to x = \topx→x=⊤，⊤→x=x\top \to x = x⊤→x=x，⊥→x=⊤\bot \to x = \top⊥→x=⊤ などがある．

ハイティング代数の性質

分配律を満たす束を分配束という．

定義. 分配束 (distributive lattice)

束 (L,⊑)(L, \sqsubseteq)(L,⊑) について、以下を満たすものを分配束と呼ぶ:

  • 任意の x,y,z∈Lx, y, z \in Lx,y,z∈L について、x∧(y∨z)=(x∧y)∨(x∧z)x \land (y \lor z) = (x \land y) \lor (x \land z)x∧(y∨z)=(x∧y)∨(x∧z)
  • 任意の x,y,z∈Lx, y, z \in Lx,y,z∈L について、x∨(y∧z)=(x∨y)∧(x∨z)x \lor (y \land z) = (x \lor y) \land (x \lor z)x∨(y∧z)=(x∨y)∧(x∨z)

なお，片方のみ成り立てば分配束になる．

定理. 分配束の条件

束 (L,⊑)(L, \sqsubseteq)(L,⊑) について、以下は同値．

  1. (L,⊑)(L, \sqsubseteq)(L,⊑) は分配束
  2. 任意の x,y,z∈Lx, y, z \in Lx,y,z∈L について、x∧(y∨z)=(x∧y)∨(x∧z)x \land (y \lor z) = (x \land y) \lor (x \land z)x∧(y∨z)=(x∧y)∨(x∧z)
  3. 任意の x,y,z∈Lx, y, z \in Lx,y,z∈L について、x∨(y∧z)=(x∨y)∧(x∨z)x \lor (y \land z) = (x \lor y) \land (x \lor z)x∨(y∧z)=(x∨y)∧(x∨z)

証明:

1 ならば 2，1 ならば 3 は自明．2 ならば 1 は，

(x∨y)∧(x∨z)=((x∨y)∧x)∨((x∨y)∧z)=x∨(x∧z)∨(y∧z)=x∨(y∧z) (x \lor y) \land (x \lor z) = ((x \lor y) \land x) \lor ((x \lor y) \land z) = x \lor (x \land z) \lor (y \land z) = x \lor (y \land z) (x∨y)∧(x∨z)=((x∨y)∧x)∨((x∨y)∧z)=x∨(x∧z)∨(y∧z)=x∨(y∧z)

より，示せる．3 ならば 1 も対称的に証明できる．

ハイティング代数は、分配束になる．

定理. ハイティング代数の分配律

ハイティング代数は，分配束である．

証明:

  • x∧y≤x∧(y∨z)x \land y \leq x \land (y \lor z)x∧y≤x∧(y∨z)
  • x∧z≤x∧(y∨z)x \land z \leq x \land (y \lor z)x∧z≤x∧(y∨z)

より，x∧(y∨z)x \land (y \lor z)x∧(y∨z) は x∧yx \land yx∧y，x∧zx \land zx∧z の上界になる．よって，(x∧y)∨(x∧z)≤x∧(y∨z)(x \land y) \lor (x \land z) \leq x \land (y \lor z)(x∧y)∨(x∧z)≤x∧(y∨z)．また，

  • x∧y≤(x∧y)∨(x∧z)x \land y \leq (x \land y) \lor (x \land z)x∧y≤(x∧y)∨(x∧z)
  • x∧z≤(x∧y)∨(x∧z)x \land z \leq (x \land y) \lor (x \land z)x∧z≤(x∧y)∨(x∧z)

より，

  • y≤(x→(x∧y)∨(x∧z))y \leq (x \to (x \land y) \lor (x \land z))y≤(x→(x∧y)∨(x∧z))
  • z≤(x→(x∧y)∨(x∧z))z \leq (x \to (x \land y) \lor (x \land z))z≤(x→(x∧y)∨(x∧z))

よって，x→(x∧y)∨(x∧z)x \to (x \land y) \lor (x \land z)x→(x∧y)∨(x∧z) は yyy，zzz の上界より y∨z≤(x→(x∧y)∨(x∧z))y \lor z \leq (x \to (x \land y) \lor (x \land z))y∨z≤(x→(x∧y)∨(x∧z))．ここから，

x∧(y∨z)≤x∧(x→(x∧y)∨(x∧z))≤(x∧y)∨(x∧z) x \land (y \lor z) \leq x \land (x \to (x \land y) \lor (x \land z)) \leq (x \land y) \lor (x \land z) x∧(y∨z)≤x∧(x→(x∧y)∨(x∧z))≤(x∧y)∨(x∧z)

よって，x∧(y∨z)=(x∧y)∨(x∧z)x \land (y \lor z) = (x \land y) \lor (x \land z)x∧(y∨z)=(x∧y)∨(x∧z) より，分配束になる．

ついでにだが、通常の有界束は分配律が成り立つとは限らない。反例は以下のもの:

この時、x∨(y∧z)=x≠⊤=(x∨y)∧(x∨z)x \lor (y \land z) = x \neq \top = (x \lor y) \land (x \lor z)x∨(y∧z)=x=⊤=(x∨y)∧(x∨z) になる。さて，束の元には補元が存在する場合がある．

定義. 補 (complement)

有界束 (L,⊑)(L, \sqsubseteq)(L,⊑) の元 x∈Lx \in Lx∈L について，

  • x∧y≤⊥x \land y \leq \botx∧y≤⊥
  • ⊤≤x∨y\top \leq x \lor y⊤≤x∨y

を満たす y∈Ly \in Ly∈L を xxx の補と呼ぶ

⊤\top⊤ と ⊥\bot⊥ は互いに補の関係になる．なお，有界束において一般に補は一意とは限らない．例えば，上の分配律が成り立たない束において，xxx の補は yyy，zzz 両方になる．もちろん存在するとも限らない．これは，線形な束を思い描いてみるといいだろう．ところで，分配束において補は存在するなら一意に定まる．

定理. 分配束における補の一意性

有界な分配束 (L,⊑)(L, \sqsubseteq)(L,⊑) について，

  • 任意の x,y,z∈Lx, y, z \in Lx,y,z∈L について、x∧(y∨z)=(x∧y)∨(x∧z)x \land (y \lor z) = (x \land y) \lor (x \land z)x∧(y∨z)=(x∧y)∨(x∧z)
  • 任意の x,y,z∈Lx, y, z \in Lx,y,z∈L について、x∨(y∧z)=(x∨y)∧(x∨z)x \lor (y \land z) = (x \lor y) \land (x \lor z)x∨(y∧z)=(x∨y)∧(x∨z)

が成り立つ時，任意の x∈Lx \in Lx∈L についてその補は高々1つである

証明:

x∈Lx \in Lx∈L の補 y1∈Ly_1 \in Ly1​∈L，y2∈Ly_2 \in Ly2​∈L が存在する時，

y1=y1∧(y2∨x)=(y1∧y2)∨(y1∧x)=(y1∧y2)∨(y2∧x)=y2∧(y1∨x)=y2 y_1 = y_1 \land (y_2 \lor x) = (y_1 \land y_2) \lor (y_1 \land x) = (y_1 \land y_2) \lor (y_2 \land x) = y_2 \land (y_1 \lor x) = y_2 y1​=y1​∧(y2​∨x)=(y1​∧y2​)∨(y1​∧x)=(y1​∧y2​)∨(y2​∧x)=y2​∧(y1​∨x)=y2​

より，y1=y2y_1 = y_2y1​=y2​．よって，存在すれば補は一意．

ハイティング代数において補は存在するとは限らないが，ハイティング代数は分配束より存在すれば一意である．さらに，ハイティング代数においては補が存在すればその形を特定できる．

定義. 擬似補 (pseudo-complement)
ハイティング代数 (H,⊑)(H, \sqsubseteq)(H,⊑) について、その要素 x∈Hx \in Hx∈H に対し、x→⊥x \to \botx→⊥ を擬似補と言い、¬x\neg x¬x と表記する

ハイティング代数において，補が存在すればそれは ¬x\neg x¬x と一致する．

定理. 擬似補と補の一意性

ハイティング代数 (H,⊑)(H, \sqsubseteq)(H,⊑) の元 x∈Hx \in Hx∈H について，その補が存在する時，それは ¬x\neg x¬x のみである

証明:

x∈Hx \in Hx∈H の補 y∈Hy \in Hy∈H が存在する時，x∧y≤⊥x \land y \leq \botx∧y≤⊥ より y≤¬xy \leq \neg xy≤¬x である．この時，

  • x∧¬x≤⊥x \land \neg x \leq \botx∧¬x≤⊥
  • ⊤≤x∨y≤x∨¬x\top \leq x \lor y \leq x \lor \neg x⊤≤x∨y≤x∨¬x

より，¬x\neg x¬x は補である．さらに，ハイティング代数は分配束なので，補は ¬x\neg x¬x のみである．

ハイティング代数ではド・モルガン則は片方のみ成り立つ．

定理. ド・モルガン則 (De Morgan's law)

ハイティング代数 (H,⊑)(H, \sqsubseteq)(H,⊑) について、以下が成り立つ:

  • 任意の x,y∈Hx, y \in Hx,y∈H について、¬x∧¬y=¬(x∨y)\neg x \land \neg y = \neg (x \lor y)¬x∧¬y=¬(x∨y)

証明:

(x∨y)∧¬x∧¬y=(x∧¬x∧¬y)∨(y∧¬∧¬y)=⊥∨⊥≤⊥ (x \lor y) \land \neg x \land \neg y = (x \land \neg x \land \neg y) \lor (y \land \neg \land \neg y) = \bot \lor \bot \leq \bot (x∨y)∧¬x∧¬y=(x∧¬x∧¬y)∨(y∧¬∧¬y)=⊥∨⊥≤⊥

より，¬x∧¬y≤¬(x∨y)\neg x \land \neg y \leq \neg (x \lor y)¬x∧¬y≤¬(x∨y)．また，

x∧¬(x∨y)≤(x∧¬(x∨y))∨(y∧¬(x∨y))=(x∨y)∧¬(x∨y)≤⊥ x \land \neg (x \lor y) \leq (x \land \neg (x \lor y)) \lor (y \land \neg (x \lor y)) = (x \lor y) \land \neg (x \lor y) \leq \bot x∧¬(x∨y)≤(x∧¬(x∨y))∨(y∧¬(x∨y))=(x∨y)∧¬(x∨y)≤⊥

より，¬(x∨y)≤¬x\neg (x \lor y) \leq \neg x¬(x∨y)≤¬x．対称性より ¬(x∨y)≤¬y\neg (x \lor y) \leq \neg y¬(x∨y)≤¬y．よって，¬(x∨y)≤¬x∧¬y\neg (x \lor y) \leq \neg x \land \neg y¬(x∨y)≤¬x∧¬y より，¬x∧¬y=¬(x∨y)\neg x \land \neg y = \neg (x \lor y)¬x∧¬y=¬(x∨y)．

ブール代数と二重否定変換

ハイティング代数のインスタンスにブール代数がある．

定義. ブール代数 (boolean algebra)

分配束 (L,⊑)(L, \sqsubseteq)(L,⊑) で以下を満たすものを、ブール代数と呼ぶ:

  • 任意の x∈Lx \in Lx∈L について、その補が存在する
定理. ブール代数はハイティング代数

ブール代数 (B,⊑)(B, \sqsubseteq)(B,⊑) は，ハイティング代数である

証明:

x,y∈Bx, y \in Bx,y∈B について，xxx の補を aaa とする．この時，x→y=a∨yx \to y = a \lor yx→y=a∨y を示す．

x∧(a∨y)=(x∧a)∨(x∧y)=x∧y≤y x \land (a \lor y) = (x \land a) \lor (x \land y) = x \land y \leq y x∧(a∨y)=(x∧a)∨(x∧y)=x∧y≤y

であり，x∧z≤yx \land z \leq yx∧z≤y となる z∈Bz \in Bz∈B について，

z=z∧(a∨x)=(z∧a)∨(x∧z)≤a∨y z = z \land (a \lor x) = (z \land a) \lor (x \land z) \leq a \lor y z=z∧(a∨x)=(z∧a)∨(x∧z)≤a∨y

より，a∨ya \lor ya∨y は x→yx \to yx→y の条件を満たす．よって，任意の元について冪が存在するため，(B,⊑)(B, \sqsubseteq)(B,⊑) はハイティング代数．

ハイティング代数がブール代数になる条件として，以下が知られている．

定理. ブール代数の同値条件

ハイティング代数 (H,⊑)(H, \sqsubseteq)(H,⊑) について，以下は同値

  1. HHH はブール代数
  2. 任意の x∈Hx \in Hx∈H について，x∨¬x=⊤x \lor \neg x = \topx∨¬x=⊤
  3. 任意の x∈Hx \in Hx∈H について，¬¬x=x\neg \neg x = x¬¬x=x

証明:

x∈Hx \in Hx∈H について x∧¬x≤⊥x \land \neg x \leq \botx∧¬x≤⊥ であること，ブール代数がハイティング代数であることから，1 と 2 の同値性はいいと思う．

2 から 3 は，

¬¬x=¬¬x∧(x∨¬x)=(¬¬x∧x)∨(¬¬x∧¬x)=¬¬x∧x \neg \neg x = \neg \neg x \land (x \lor \neg x) = (\neg \neg x \land x) \lor (\neg \neg x \land \neg x) = \neg \neg x \land x ¬¬x=¬¬x∧(x∨¬x)=(¬¬x∧x)∨(¬¬x∧¬x)=¬¬x∧x
=(¬¬x∧x)∨(x∧¬x)=(¬¬x∨¬x)∧x=x = (\neg \neg x \land x) \lor (x \land \neg x) = (\neg \neg x \lor \neg x) \land x = x =(¬¬x∧x)∨(x∧¬x)=(¬¬x∨¬x)∧x=x

より示せる．3 から 2 は，ハイティング代数で適用できるド・モルガン則を使用して，

x∨¬x=¬¬(x∨¬x)=¬(¬x∧¬¬x)=¬⊥=⊤ x \lor \neg x = \neg \neg (x \lor \neg x) = \neg (\neg x \land \neg \neg x) = \neg \bot = \top x∨¬x=¬¬(x∨¬x)=¬(¬x∧¬¬x)=¬⊥=⊤

より示せる．

ブール代数では，2つめのド・モルガン則も示せる．

定理. ド・モルガン則 (De Morgan's law)

ブール代数 (B,⊑)(B, \sqsubseteq)(B,⊑) について、以下が成り立つ:

  • 任意の x,y∈Hx, y \in Hx,y∈H について、¬x∨¬y=¬(x∧y)\neg x \lor \neg y = \neg (x \land y)¬x∨¬y=¬(x∧y)

証明:

ブール代数では二重否定の除去ができることから，ハイティング代数上で成り立つド・モルガン則を使用して，

¬x∨¬y=¬¬(¬x∨¬y)=¬(¬¬x∧¬¬y)=¬(x∧y) \neg x \lor \neg y = \neg \neg (\neg x \lor \neg y) = \neg (\neg \neg x \land \neg \neg y) = \neg (x \land y) ¬x∨¬y=¬¬(¬x∨¬y)=¬(¬¬x∧¬¬y)=¬(x∧y)

のように示せる．

さて，ハイティング代数からブール代数への変換方法として，二重否定変換というものが知られている．これは，名前の通り全ての元に二重否定をかますような変換だ．それを示すため，まず準備として補題を用意する．

補題.

ハイティング代数 (H,⊑)(H, \sqsubseteq)(H,⊑) について，以下が成り立つ:

圏論的視点

今までは通常の束論の議論でハイティング代数を取り扱ってきたが，圏論的な議論もしておく．まず，半順序での最大下界，最小上界は，poset での直積，直和になる．これは，定義を比べてみれば分かると思う．そして，最小元，最大元は始対象，終対象に相当する．よって，有界束とは，finitely complete かつ cocomplete な poset のことになる [1] ．

定義. 有界束

poset CCC が有界束とは，以下を満たすこと

  • 有限積を持つ (finitely complete)
  • 有限余積を持つ (finitely cocomplete)

ハイティング代数は，有界束のなかで CCC (Cartesian Closed Category) になるものである．すなわち，冪も持つような poset になる．
