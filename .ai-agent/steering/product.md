# プロダクトビジョン

## ミッション

技術的な知見や学びを、自分の手で管理できるプラットフォーム上で継続的に発信すること。

## ターゲットユーザー

- 主要な読者: プログラミングに関心のある日本語話者の技術者
- 著者自身: 学びのアウトプットと整理の場

## ユースケース

- 技術記事の執筆・公開（reStructuredText / Markdown）
- 数式を含む専門的な記事の公開（KaTeX レンダリング）
- RSS フィードによる記事配信
- GitHub Pages での静的サイトホスティング

## 主要機能

- 記事の執筆・管理（年/月別アーカイブ）
- KaTeX による数式レンダリング
- ダーク/ライトテーマ切り替え
- 脚注のツールチップ表示（Tippy.js）
- レスポンシブデザイン
- Sentry によるエラー監視
- サイトマップ・RSS フィード生成
- 関連記事表示
- CC-BY-SA 4.0 ライセンスでの記事公開
